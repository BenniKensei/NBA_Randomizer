## 📥 How to Use These Resources

### For SVG Icons:
```typescript

// Then use in component:

```

### For Background Images:
```css
/* In your CSS file */
.basketball-bg {
  background-image: url('/images/court-texture.jpg');
  background-size: cover;
  opacity: 0.05;
}
```
